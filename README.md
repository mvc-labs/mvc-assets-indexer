
# mvc-assets-indexer

The index service for P2PKH and MetaContract FT assets on [MicroVisionChain](https://www.microvisionchain.com/).

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
